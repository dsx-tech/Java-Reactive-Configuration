    Architecture

    Architecture concepts

    ConfigManager is abstract class that should be implemented by configuration object (or a singleton class for Java) and provide declarative DSL for configuration structure declaration just like Konfig library. This object stores a link to channel, that takes stream of RawProperties[] filled by ConfigSources, and emmits changes to flow of properties.

    Each registered property provides a Reloadable instance, that can be fetched by get method of ConfigManager.
